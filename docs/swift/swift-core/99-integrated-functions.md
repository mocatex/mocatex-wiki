# useful Integrated Functions

many useful functions (AND properties) are already integrated in Swift. Here are some of them:

| Function               | Description                                             |
| ---------------------- | ------------------------------------------------------- |
| `.contains("")`        | checks if a string or array contains a specific element |
| `.count`               | returns the number of elements in a array               |
| `.hasPrefix("")`       | checks if a string has a specific prefix                |
| `.hasSuffix("")`       | checks if a string has a specific suffix                |
| `.isMultiple(of: Int)` | checks if a number is multiple of Int                   |
| `.lowercased()`        | returns a string in lowercase                           |
| `.reversed()`          | reverses a string or array                              |
| `.sorted()`            | sorts a string or array                                 |
| `.uppercased()`        | returns a string in uppercase                           |
| `.remove(at: Int)`     | removes an element at a specific index                  |
| `.removeAll()`         | removes all elements from a string or array             |
| `.isEmpty`             | checks if a string or array is empty                    |
